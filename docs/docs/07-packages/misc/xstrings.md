# Xstrings (xstrings)

The `xstrings` package provides utilities used by Ignite CLI.

For full API details, see the
[`xstrings` Go package documentation](https://pkg.go.dev/github.com/ignite/cli/v29/ignite/pkg/xstrings).

